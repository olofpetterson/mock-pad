    **RequestLogListView.swift** -- Main log list view:
    - `@Query(sort: \RequestLog.timestamp, order: .reverse) private var logs: [RequestLog]`
    - `@Environment(EndpointStore.self) private var endpointStore`
    - `@Environment(ServerStore.self) private var serverStore`
    - `@State private var activeMethodFilters: Set<String> = []`
    - `@State private var activeStatusFilters: Set<String> = []`
    - `@State private var searchText: String = ""`
    - Computed `filteredLogs: [RequestLog]` applying AND logic:
      - Method: `activeMethodFilters.isEmpty || activeMethodFilters.contains(log.method.uppercased())`
      - Status: `activeStatusFilters.isEmpty || activeStatusFilters.contains(statusCategory(log.responseStatusCode))`
      - Search: `searchText.isEmpty || log.path.localizedCaseInsensitiveContains(searchText)`
    - Private `statusCategory(_ code: Int) -> String`: 200..<300 -> "2xx", 400..<500 -> "4xx", 500..<600 -> "5xx", default -> "other"
    - Body structure:
      - VStack(spacing: 0):
        1. LogFilterChipsView(activeMethodFilters: $activeMethodFilters, activeStatusFilters: $activeStatusFilters) -- top filter bar
        2. If filteredLogs.isEmpty: contextual empty state (see below)
        3. Else: List(filteredLogs) with NavigationLink to RequestDetailView(log: log) wrapping RequestLogRowView(log: log)
      - `.searchable(text: $searchText, prompt: "Filter by path...")` on the VStack or List
      - `.navigationTitle("Request Log")`
      - `.toolbar`: ToolbarItem(placement: .topBarTrailing) with Button calling `endpointStore.clearLog()` with `Image(systemName: "trash")` -- disabled when logs is empty
      - `.listStyle(.plain)`, `.scrollContentBackground(.hidden)`, `.background(MockPadColors.background)`
      - List row styling: `.listRowBackground(MockPadColors.background)`, `.listRowSeparator(.hidden)`

    **Contextual empty states** (inside a VStack with Spacer centering):
    - If `!serverStore.isRunning && logs.isEmpty`: "Start the server to begin logging requests." with `Image(systemName: "server.rack")` in textMuted
    - Else if `logs.isEmpty` (server running, no requests): "Waiting for requests..." with `Image(systemName: "antenna.radiowaves.left.and.right")` in textMuted
    - Else if `filteredLogs.isEmpty` (filters hiding all): "No requests match your filters." with a "Clear Filters" button that sets `activeMethodFilters = []`, `activeStatusFilters = []`, `searchText = ""`
    - Empty state text: `.font(MockPadTypography.bodySmall)`, `.foregroundColor(MockPadColors.textMuted)`, `.multilineTextAlignment(.center)`
    - Empty state icon: `.font(.system(size: 40))`, `.foregroundColor(MockPadColors.textDisabled)`, `.padding(.bottom, 8)`
