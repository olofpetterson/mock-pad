    Create RequestDetailView accepting `let log: RequestLog` parameter.

    **State:**
    - `@Environment(ServerStore.self) private var serverStore` -- for baseURL in cURL generation
    - `@State private var showCopiedFeedback = false`

    **Layout:** ScrollView containing VStack(alignment: .leading, spacing: MockPadMetrics.panelSpacing):

    **Section 1: Request Summary** (always visible, not collapsible)
    - Panel background (MockPadColors.panel, cornerRadius MockPadMetrics.cornerRadius)
    - HStack: Method badge (using `.methodBadgeStyle()`), path text (`.endpointPathStyle()`), Spacer, status code (`.font(MockPadTypography.statusCode)`, colored by `statusCodeColor`)
    - Below: HStack with timestamp (`log.timestamp.formatted(date: .abbreviated, time: .standard)`) and response time (`String(format: "%.1fms", log.responseTimeMs)`) in textMuted

    **Section 2: Request Details** (DisclosureGroup, expanded by default)
    - Section label: `Text("> REQUEST_").blueprintLabelStyle()`
    - **Query Parameters** (only if log.queryParameters is non-empty):
      - Sub-label: "Query Parameters" in textMuted
      - ForEach over sorted queryParameters keys: `Text("\(key): \(value)")` in monoSmall font
    - **Request Headers** (only if log.requestHeaders is non-empty):
      - Sub-label: "Headers" in textMuted
      - ForEach over sorted requestHeaders keys: `Text("\(key): \(value)")` in monoSmall font
    - **Request Body** (only if log.requestBody is non-nil and non-empty):
      - Sub-label: "Body" in textMuted
      - Text(log.requestBody!) in codeEditorStyle, inside a panel2 background with padding

    **Section 3: Response Details** (DisclosureGroup, expanded by default)
    - Section label: `Text("> RESPONSE_").blueprintLabelStyle()`
    - **Matched Endpoint** (only if log.matchedEndpointPath is non-nil):
      - Text("Matched: \(log.matchedEndpointPath!)") in monoSmall font, textAccent color
    - **Response Headers** (only if log.responseHeaders is non-empty):
      - Sub-label: "Headers" in textMuted
      - ForEach over sorted responseHeaders keys: `Text("\(key): \(value)")` in monoSmall font
    - **Response Body** (only if log.responseBody is non-nil and non-empty):
      - Sub-label: "Body" in textMuted
      - Text(log.responseBody!) in codeEditorStyle, inside a panel2 background with padding

    **Section 4: Copy as cURL button**
    - Button with `Label("Copy as cURL", systemImage: "doc.on.doc")`
    - On tap:
      ```swift
      let curl = CurlGenerator.generate(
          method: log.method,
          path: log.path,
          headers: log.requestHeaders,
          body: log.requestBody,
          baseURL: serverStore.serverURL
      )
      UIPasteboard.general.string = curl
      showCopiedFeedback = true
      ```
    - Style: full-width button, accent color background, MockPadColors.background text
    - `.buttonStyle(.plain)`, padding, cornerRadius

    **Copied feedback:** Overlay or inline text "Copied!" that appears when `showCopiedFeedback` is true. Use `.task(id: showCopiedFeedback)` to auto-dismiss after 2 seconds:
    ```swift
    .onChange(of: showCopiedFeedback) { _, newValue in
        if newValue {
            Task {
                try? await Task.sleep(for: .seconds(2))
                showCopiedFeedback = false
            }
        }
    }
    ```

    **Navigation:** `.navigationTitle("\(log.method) \(log.path)")`, `.navigationBarTitleDisplayMode(.inline)`
    **Background:** `.background(MockPadColors.background)`

    **DisclosureGroup styling:** Use `.tint(MockPadColors.accent)` for the disclosure arrow color. Each DisclosureGroup content in a VStack(alignment: .leading, spacing: 8) with `.padding(.top, 4)`.

    **IMPORTANT layout safety:** Do NOT use Color as a standalone view inside ScrollView. Use `.background()` modifier on actual content views with shapes, never bare Color views. This prevents infinite layout loops per project memory.
