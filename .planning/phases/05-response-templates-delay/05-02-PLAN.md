    Environment: `@Environment(\.modelContext) private var modelContext`, `@Environment(ProManager.self) private var proManager`.

    State: `@State private var showingSaveSheet = false`.

    Layout structure:
    1. **Built-in Templates section** -- `Section` with header `"> TEMPLATES_"` using `.blueprintLabelStyle()`. ForEach over `BuiltInTemplates.all`. Each row is a Button containing an HStack with:
       - `Image(systemName: template.icon)` tinted accent
       - `Text(template.name)` using `.font(MockPadTypography.monoSmall)`
       - `Spacer()`
       - `Text("\(template.statusCode)")` in a rounded badge with status color from `MockPadColors.statusColor(template.statusCode)`
       Button action calls `applyTemplate(template)`.
       `.listRowBackground(MockPadColors.panel)`

    2. **Custom Templates section** (only if `!customTemplates.isEmpty` OR `proManager.isPro`) -- `Section` with header `"> CUSTOM TEMPLATES_"` using `.blueprintLabelStyle()`.
       - If `proManager.isPro`: Show ForEach over `customTemplates` with same row pattern as built-in (but using custom template fields). Each row is a Button that calls `applyCustomTemplate(ct)`. Add `.onDelete` modifier for swipe-to-delete via `deleteCustomTemplate(at:)`.
       - Below the list: Button "Save Current as Template" that sets `showingSaveSheet = true`. Style: `Image(systemName: "plus.circle")` + text, tinted accent.
       - If `!proManager.isPro`: Show a Text "PRO" label indicating this is a PRO feature.
       `.listRowBackground(MockPadColors.panel)`

    3. `.sheet(isPresented: $showingSaveSheet)` presenting `SaveTemplateSheet(endpoint: endpoint)`.

    Helper methods:
    - `applyTemplate(_ template: BuiltInTemplates.Template)`: Sets `endpoint.responseStatusCode = template.statusCode`, `endpoint.responseBody = template.responseBody`, `endpoint.responseHeaders = template.responseHeaders`. Calls `onApplied()`.
    - `applyCustomTemplate(_ ct: ResponseTemplate)`: Same pattern but from ResponseTemplate fields. Calls `onApplied()`.
    - `deleteCustomTemplate(at offsets: IndexSet)`: Deletes from modelContext, saves.

    Props: `let endpoint: MockEndpoint`.
    Environment: `@Environment(\.modelContext) private var modelContext`, `@Environment(\.dismiss) private var dismiss`.
    State: `@State private var templateName: String = ""`.

    Layout: NavigationStack wrapping a Form with:
    - Section with header "> TEMPLATE NAME_" containing TextField "Template name" bound to $templateName.
    - Section "Preview" showing read-only: status code, first 3 lines of response body (truncated), header count.
    - `.navigationTitle("Save Template")`, `.navigationBarTitleDisplayMode(.inline)`.
    - Toolbar: leading Cancel button (dismiss), trailing Save button (calls `saveTemplate()`, disabled if templateName is empty after trimming).

    `saveTemplate()`: Create `ResponseTemplate(name: templateName.trimmingCharacters(in: .whitespaces), statusCode: endpoint.responseStatusCode, responseBody: endpoint.responseBody, responseHeaders: endpoint.responseHeaders)`. Insert into modelContext, save, dismiss.

    Initialize `templateName` in `.onAppear` to endpoint.path (pre-fill with endpoint path for convenience).
