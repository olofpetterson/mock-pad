Purpose: Deliver the server-side delay simulation (RESP-06 success criteria 5) so clients experience realistic latency when testing against mock endpoints.
Output: Updated EndpointMatcher with delay in data flow, updated MockServerEngine with Task.sleep before response send.

