Body layout inside NavigationStack > ScrollView > VStack(spacing: MockPadMetrics.paddingLarge):

**Header section:**
- Large lock.shield.fill icon (42pt) with proGradientEnd background circle, accent foreground
- "MockPad PRO" title using MockPadTypography.title, textPrimary color
- "Unlock the full power of MockPad" subtitle using MockPadTypography.bodySmall, textMuted color

**Feature list section:**
- ForEach over features array (use index as id since it is a fixed array)
- Each row: HStack with icon (accent color, frame 24x24), title text (monoSmall font, textPrimary), Spacer, checkmark icon (accent, opacity 0.6)
- Each row has padding, panel background, cornerRadius, proBorder overlay
- Use MockPadMetrics.paddingSmall between items

**Price section:**
- If proManager.product is non-nil, show:
  - product.displayPrice in large title font (accent color)
  - "One-time purchase" in bodySmall font, textMuted color
  - "No subscription. Pay once, unlock forever." in bodySmall font, textMuted color
- If proManager.product is nil, show:
  - ProgressView with "Loading..." text, textMuted color

**Purchase button:**
- Full-width button with "Unlock PRO" label
- Uses MockPadTypography.button font
- .borderedProminent style, accent tint
- Disabled when: product is nil OR purchaseState is .purchasing
- On tap: `Task { await proManager.purchase() }`
- When purchaseState is .purchasing, show ProgressView() instead of text
- Use .buttonStyle(.borderedProminent) and .controlSize(.large)

**Error display:**
- If purchaseState is .failed(let message), show Text(message) in status5xx color, bodySmall font, below the purchase button

**Pending display:**
- If purchaseState is .pending, show "Purchase pending approval." in status4xx color, bodySmall font

**Restore button:**
- Below purchase button: `Button("Restore Purchase") { Task { await proManager.restorePurchases() } }`
- Style: .plain button, accent color, bodySmall font
- Add small text below: "Previously purchased? Tap to restore." in textMuted, logTimestamp font

**Auto-dismiss on purchase:**
- Add `.onChange(of: proManager.isPro)` modifier. When isPro becomes true, call `dismiss()`.

**Navigation setup:**
- .navigationTitle("MockPad PRO")
- .navigationBarTitleDisplayMode(.inline)
- .toolbar with dismiss button (X icon or "Close" text) at .cancellationAction
- .scrollContentBackground(.hidden) if applicable
- Background: LinearGradient from proGradientStart to proGradientEnd, .ignoresSafeArea()

**Padding:**
- Content VStack has .padding(.horizontal, MockPadMetrics.panelPadding)
- Bottom padding for scroll content

Theme the entire view with MockPad's Slate Blueprint design system. Use existing color, typography, and metrics tokens exclusively. Do NOT introduce new design tokens.
