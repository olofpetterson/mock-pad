Otherwise, try `CollectionExporter.exportDocument(endpoints: filteredEndpoints, collectionName: selectedCollection)`, set `exportDocument` and `showExporter = true`. Catch errors and show in importError alert.
    2. **ShareLink** (PRO) — Only show when exportDocument is not nil OR provide a dynamic ShareLink. Better approach: Use a separate `Button("Share Collection", systemImage: "square.and.arrow.up.on.square")` that prepares the export data, then triggers a share. Actually, ShareLink needs the item at build time. Use this pattern instead: create a computed `exportFileForShare: MockPadExportFile?` that returns nil if no endpoints to share. Use `if let file = exportFileForShare { ShareLink(item: file, preview: SharePreview("MockPad Endpoints", image: Image(systemName: "doc.text"))) }`. Gate behind isPro with opacity pattern.

    Actually, simpler approach for Share: Place the ShareLink inside the menu. Compute the share item on-demand. The ShareLink item must be available at view build time. Use a `ShareLink` with the item produced by encoding `filteredEndpoints`. If PRO: show ShareLink. If not PRO: show disabled button.

    Revised approach — Use a `Menu` with three items:
    1. `Button("Export to File", systemImage: "square.and.arrow.down")` — PRO-gated. Prepares MockPadDocument from filteredEndpoints, sets showExporter = true.
    2. A `ShareLink("Share", item: shareFile, preview: SharePreview(...))` — PRO-gated. `shareFile` is a computed MockPadExportFile from current filteredEndpoints.
    3. `Button("Import from File", systemImage: "square.and.arrow.down.on.square")` — Available to all users. Sets showImporter = true.

    For PRO gating inside Menu: use `.disabled(!proManager.isPro)` on the export button. For ShareLink, conditionally show it only when isPro, otherwise show a disabled button placeholder.

    **fileExporter modifier** — Attach to the view:
    ```
    .fileExporter(
        isPresented: $showExporter,
        document: exportDocument,
        contentType: .json,
        defaultFilename: (selectedCollection ?? "endpoints") + ".json"
    ) { result in
        if case .failure(let error) = result {
            importError = error.localizedDescription
            showImportError = true
        }
        exportDocument = nil
    }
    ```

    **fileImporter modifier** — Attach to the view:
    ```
    .fileImporter(
        isPresented: $showImporter,
        allowedContentTypes: [.json],
        allowsMultipleSelection: false
    ) { result in
        // Handle security-scoped file access
        guard let url = (try? result.get())?.first else { return }
        let accessing = url.startAccessingSecurityScopedResource()
        defer { if accessing { url.stopAccessingSecurityScopedResource() } }
        do {
            let data = try Data(contentsOf: url)
            let export = try CollectionImporter.parse(data: data)
            pendingImport = export
            showImportPreview = true
        } catch {
            importError = error.localizedDescription
            showImportError = true
        }
    }
    ```

    **Error alert** — `.alert("Import Error", isPresented: $showImportError) { Button("OK", role: .cancel) {} } message: { Text(importError ?? "Unknown error") }`

    **PRO alert for export** — `.alert("PRO Required", isPresented: $showExportProAlert) { Button("OK", role: .cancel) {} } message: { Text("Upgrade to PRO to export endpoint collections.") }`

    **Import preview sheet** — `.sheet(isPresented: $showImportPreview) { if let export = pendingImport { ImportPreviewSheet(export: export) } }`

    Add `import UniformTypeIdentifiers` at top.

    Add `debouncedSyncEngine()` call after import completes (via the ImportPreviewSheet's dismiss callback or onDismiss on the sheet).

    **ImportPreviewSheet.swift** (NEW) — Create a sheet view for previewing and confirming an import.

    Properties:
    - `let export: MockPadExport`
    - `@Environment(EndpointStore.self) private var endpointStore`
    - `@Environment(ProManager.self) private var proManager`
    - `@Environment(\.dismiss) private var dismiss`
    - `@State private var showDuplicateResolution = false`
    - `@State private var duplicateCount = 0`
    - `@State private var showLimitAlert = false`

    Body: `NavigationStack` wrapping a `Form`:

    **Header section** — Show import summary:
    - Collection name (or "Ungrouped" if nil): `export.collectionName ?? "Ungrouped"`
    - Endpoint count: `"\(export.endpoints.count) endpoint(s)"`
    - Exported date: formatted with `.dateTime`

    **Endpoint preview section** — `ForEach(export.endpoints, id: \.path)` showing each endpoint as a row with method badge and path. Use `HStack` with method text in `MockPadColors.methodColor(for:)` and path in mono font. If multiple endpoints share same path, use a combined id like `"\(ep.httpMethod) \(ep.path)"`.

    **Import button** — A prominent `Button("Import \(export.endpoints.count) Endpoint(s)")` at the bottom. On tap:
    1. Check PRO limit: `proManager.canImportEndpoints(currentCount: endpointStore.endpointCount, importCount: export.endpoints.count)`. If not, set `showLimitAlert = true` and return.
    2. Check for duplicates: `let duplicates = CollectionImporter.findDuplicates(imported: export.endpoints, existing: endpointStore.endpoints)`.
    3. If `duplicates.isEmpty` — directly call `endpointStore.importEndpoints(export.endpoints, collectionName: export.collectionName, resolution: .importAsNew)` and `dismiss()`.
    4. If duplicates found — set `duplicateCount = duplicates.count` and `showDuplicateResolution = true`.

    **Duplicate resolution** — `.confirmationDialog("Duplicates Found", isPresented: $showDuplicateResolution, titleVisibility: .visible)` with message `"\(duplicateCount) of \(export.endpoints.count) endpoints already exist."` and actions:
    - `Button("Skip Duplicates") { endpointStore.importEndpoints(export.endpoints, collectionName: export.collectionName, resolution: .skip); dismiss() }`
    - `Button("Replace Existing") { endpointStore.importEndpoints(export.endpoints, collectionName: export.collectionName, resolution: .replace); dismiss() }`
    - `Button("Import as New") { endpointStore.importEndpoints(export.endpoints, collectionName: export.collectionName, resolution: .importAsNew); dismiss() }`
    - `Button("Cancel", role: .cancel) {}`

    **PRO limit alert** — `.alert("Endpoint Limit", isPresented: $showLimitAlert) { Button("OK", role: .cancel) {} } message: { Text("Importing \(export.endpoints.count) endpoints would exceed the free limit of \(ProManager.freeEndpointLimit) endpoints. Upgrade to PRO for unlimited endpoints.") }`

    **Navigation** — `.navigationTitle("Import Preview")`, `.navigationBarTitleDisplayMode(.inline)`, toolbar cancel button: `ToolbarItem(placement: .cancellationAction) { Button("Cancel") { dismiss() } }`.

    **Styling** — `.scrollContentBackground(.hidden)`, `.background(MockPadColors.background)`, `.listRowBackground(MockPadColors.panel)` on sections.
