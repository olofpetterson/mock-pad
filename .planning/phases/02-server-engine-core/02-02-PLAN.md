Create MockServerEngine.swift with `import Foundation` and `import Network`.

**Actor declaration:**
```swift
actor MockServerEngine {
    private var listener: NWListener?
    private var connectionMap: [ObjectIdentifier: NWConnection] = [:]
    private var endpoints: [EndpointSnapshot] = []
    private var corsEnabled: Bool = true
    private let maxConnections = 50
    private let maxRequestSize = 64 * 1024  // 64KB

    var onRequestLogged: (@Sendable (RequestLogData) -> Void)?

    private(set) var isListening: Bool = false
    private(set) var actualPort: UInt16 = 0
}
```

**Public API (3 methods):**

1. `func start(port: UInt16, endpoints: [EndpointSnapshot], corsEnabled: Bool) throws` --
   - Guard not already listening (throw .alreadyRunning)
   - Create NWParameters.tcp with acceptLocalOnly=true, allowLocalEndpointReuse=true
   - Create NWEndpoint.Port from rawValue (throw .invalidPort if nil)
   - Create new NWListener (MUST be new instance each time -- cannot restart cancelled listener per SR-13918)
   - Set stateUpdateHandler: `{ [weak self] state in Task { await self?.handleListenerState(state) } }`
   - Set newConnectionHandler: `{ [weak self] conn in Task { await self?.handleNewConnection(conn) } }`
   - Start listener on DispatchQueue(label: "com.mockpad.server")
   - Store self.listener, self.endpoints, self.corsEnabled

2. `func stop()` --
   - Cancel listener, set listener=nil
   - Cancel all connections in connectionMap, clear connectionMap
   - Set isListening=false, actualPort=0

3. `func updateEndpoints(_ endpoints: [EndpointSnapshot])` --
   - Replace self.endpoints with new snapshots (called from ServerStore when user edits endpoints)

**Private handler methods:**

4. `private func handleListenerState(_ state: NWListener.State)` --
   - `.ready`: set isListening=true, extract actual port from listener?.port?.rawValue ?? 0, set actualPort
   - `.failed(let error)`: stop(), (error is logged but not thrown -- already in callback)
   - `.cancelled`: isListening=false
   - default: break

5. `private func handleNewConnection(_ connection: NWConnection)` --
   - Guard connectionMap.count < maxConnections (else: send 503 response via buildServiceUnavailable, cancel connection, return)
   - let id = ObjectIdentifier(connection)
   - Store connection in connectionMap[id]
   - Set connection stateUpdateHandler for cleanup: on .failed/.cancelled, remove from connectionMap
   - Start connection on same DispatchQueue(label: "com.mockpad.connection.\(id.hashValue)")
   - Call receiveRequest(connection: connection, id: id)

6. `private func receiveRequest(connection: NWConnection, id: ObjectIdentifier)` --
   - connection.receive(minimumIncompleteLength: 1, maximumLength: maxRequestSize) { [weak self] data, _, _, error in Task { await self?.handleReceivedData(data: data, connection: connection, id: id) } }

7. `private func handleReceivedData(data: Data?, connection: NWConnection, id: ObjectIdentifier)` --
   - let startTime = CFAbsoluteTimeGetCurrent()
   - Guard data exists and is not empty (else: send 400 bad request, cleanup, return)
   - Parse with HTTPRequestParser.parse(data:) -- if nil, send 400, cleanup, return
   - Check for OPTIONS method: if parsedRequest.method == "OPTIONS", send HTTPResponseBuilder.buildPreflightResponse(corsEnabled:), cleanup, return
   - Match with EndpointMatcher.match(method:path:endpoints:) converting self.endpoints to EndpointData tuples
   - Switch on MatchResult:
     - .matched: build response with HTTPResponseBuilder.build(statusCode:headers:body:corsEnabled:) using matched endpoint's data. Add endpoint's custom responseHeaders.
     - .notFound: build 404 JSON error response `{"error":"Not Found","path":"..."}` with Content-Type: application/json
     - .methodNotAllowed: build 405 JSON error response `{"error":"Method Not Allowed","allowed":["GET",...]}` with Content-Type: application/json and Allow header (comma-separated allowed methods)
   - Send response data via connection.send(content:completion:)
   - Calculate responseTimeMs = (CFAbsoluteTimeGetCurrent() - startTime) * 1000
   - Create RequestLogData and call onRequestLogged?(logData)
   - After send completion, cancel connection and remove from connectionMap

**CRITICAL implementation notes:**
- All NWListener/NWConnection callbacks run on DispatchQueue. MUST bridge to actor via `Task { [weak self] in await self?.method() }`.
- Use `[weak self]` in ALL closures registered on NWListener/NWConnection to prevent retain cycles.
- NWConnection is NOT Hashable -- use ObjectIdentifier as dictionary key.
- Connection stateUpdateHandler needs `[weak self]` and bridges to actor: `Task { await self?.removeConnection(id) }`
- The send completion handler should cancel the connection (HTTP/1.0 close-after-response).
